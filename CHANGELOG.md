# Changelog

## Card Builder V1.1 - 2026-04-08

视觉与交互升级版本，重做顶部导航、表单控件、模板切换与实时预览交互，并补齐组件学习 playground。

### Included

- 顶部导航改为固定通栏，增加模板切换器、结构线与交点装饰，整体视觉对齐新版设计语言。
- 左侧编辑区重构为更统一的白色表单卡片体系，按钮、输入框、下拉框与锁定/显示控制统一样式。
- 支持中文名、英文名显示/隐藏切换，并在预览中保持姓名区对齐逻辑。
- 电话区号、网站与地址地区统一为自定义 `Select` 交互；邮箱改为前缀输入与固定 `@shoplazza.com` 后缀。
- Shoplazza 模板「公司」改为下拉选择（三项预设或空），与其它下拉样式一致。
- 实时预览区支持无二维码时的可点击上传占位框，并保留名片尺寸说明。
- `Subotiz` 模板入口改为弹窗提示“模板搭建中...”，避免误切换。
- 新增并完善 `/playground` 组件学习页面，用于验证当前 UI 组件与样式系统。

### Notes

- 当前正式可编辑模板仍为 `Shoplazza`。
- 当前版本不包含印刷版导出。

## Card Builder V1.0 - 2026-03-30

首个可交付版本，完成名片编辑、预览和电子版 PDF 导出主链路。

### Included

- 提供 `Shoplazza 名片` 默认模板，支持中文名、英文名、职位、电话、邮箱、网站、公司、地址地区与二维码内容编辑。
- 默认模板接入鸿蒙字体与 `PPRightGrotesk-WideRegular`，并同步到页面预览与电子版 PDF 导出。
- 支持二维码图片解码与手动填写，预览和导出均使用统一尺寸的矢量二维码。
- 提供服务端生成的矢量 PDF 电子版导出方案，文本可复制搜索，内置 logo 与二维码保持矢量。
- 模板 A 完成 Shoplazza 默认样式与地址地区预设。
- 模板 B 入口保留，但当前以 `Subotiz 名片（搭建中）` 形式禁用展示。
- 左侧编辑区、顶部标题栏与右侧预览区的基础交互和固定布局已完成。

### Notes

- 当前版本不包含印刷版导出。
- 当前版本仅开放 `Shoplazza 名片` 模板作为可编辑模板。
